## Smart contract

- Minter:
	* mints the smart contract
	* transfers NFT to the smart contract
- Printer:
	* receives the printer code
	* prints the wearable with the receiver code
- Receiver:
	* receives the receiver code
	* enters:
		* receiver code
		* printer code
	* receives NFT
